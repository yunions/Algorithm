- Stack, Queue : 기본적인 자료구조 중 하나 → 데이터를 넣곡 뽑는 방향이 다름

## (1) Stack

: 데이터를 임시 저장할 때 사용하는 자료구조

- 데이터의 입력과 출력 순서 :  **LIFO**(last in first out, 후입선출방식)  like ctrl + z

### Stack basic

- **stk** : 스택 배열(list)
- **capacity** : 스택의 최대 크기(int) = len(stk)
- **ptr** : 스택 포인터, 스택에 쌓여 있는 데이터의 수

### Stack function

- **push** : stack 구조에 데이터를 넣는 함수(가장 top에 들어감)
- **pop** : stack 구조에서 데이터를 꺼내는 함수(가장 top을 꺼냄)
- **peak** : top의 데이터 값 확인(= 가장 최근 push 데이터 확인)
